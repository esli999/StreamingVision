# StreamingVision

## Environment

Conda environment: `streamingvision` (Python 3.11)

Dependencies:
- `torch==2.6.0+cu124`
- `torchvision==0.21.0+cu124`
- `timm==1.0.19`
- `jax[cuda12]==0.5.3`

### Setup

```bash
conda create -n streamingvision python=3.11 -y
conda activate streamingvision
pip install -r requirements.txt
```
